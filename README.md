# 📊 Life on the Hedge Fund — Portfolio Analytics Dashboard

Institutional-style portfolio analytics terminal built in Python.

---

## 🧠 Project Overview

**Life on the Hedge Fund** is an academic portfolio project developed at
**Trinity College Dublin (#1 in Ireland)**.

The objective is to replicate a **buy-side / asset management analytics environment**,
with a strong focus on:

* Portfolio performance tracking
* Risk analytics
* Benchmark comparison
* Scenario analysis
* Investment intelligence

---

## 📌 Portfolio Characteristics

* 💰 Initial AUM: ~$50,000
* 📈 Strategy: Concentrated US Equity Portfolio
* ⚡ Style: High-conviction / high-beta / thematic growth
* 🔁 Rebalancing: None since inception
* 🎯 Benchmark: Nasdaq-100 (QQQ)
* 🕒 Horizon: 15 years

---

## ⚙️ Architecture

This project is built with a **Python-first architecture**:

* Python is the **single source of truth**
* All analytics are computed server-side
* HTML dashboard is fully generated by Python
* No fragile browser-side data fetching

---

## 🔄 Automation

The dashboard is automatically rebuilt using **GitHub Actions**:

* ⏱️ Runs every hour (weekdays)
* 📊 Fetches latest market data via `yfinance`
* 🧮 Recomputes all portfolio metrics
* 🌐 Updates the public dashboard

---

## 📊 Features

### Core Analytics

* NAV & P&L tracking
* Portfolio vs benchmark (QQQ / SPY)
* Alpha, Beta, Sharpe, Sortino
* Drawdown analysis
* Rolling volatility & beta

### Risk Metrics

* VaR / CVaR
* Downside deviation
* Information ratio
* Tracking error
* Jensen’s alpha
* Treynor ratio

### Portfolio Breakdown

* Position-level performance
* Sector allocation
* Thematic exposure
* Risk buckets
* Contribution analysis

### Advanced

* Monte Carlo scenario projections
* Stress testing engine
* Portfolio intelligence (automated insights)
* News aggregation (Python-based)
